Autonomous Car Algorithm Development

Dates: May 2024

Class: ECE484 - Final Project

Successfully implemented an autonomous racing solution for the GRAIC competition using ROS (Robot Operating System) and CARLA simulation environment. The project focused on advanced computer vision analysis, intelligent pathfinding, and sophisticated obstacle avoidance systems, all while exceeding the minimum required lap time performance metrics.

Advanced Pathfinding and Optimization

Implemented RRT (Rapidly-exploring Random Trees) pathfinding algorithm with custom optimizations for dynamic racing environments. Enhanced the base algorithm with gradient descent optimization for waypoint generation, resulting in an optimal racing line with minimal curvature. This innovation significantly improved cornering speeds while maintaining the vehicle's agility for obstacle avoidance.

Intelligent Control Systems

Developed a sophisticated PID control loop with custom modifications for intelligent brake management. The system implements light braking techniques rather than traditional aggressive braking approaches, allowing for:

  • Enhanced momentum conservation through corners
  • Smoother transitions between waypoints
  • Improved overall vehicle stability
  • Superior lap times compared to standard implementations

Computer Vision and Obstacle Management

Utilized Python to create a comprehensive computer vision system in ROS that handles:

  • Real-time obstacle detection and tracking
  • Dynamic path adjustment based on environmental conditions
  • Predictive avoidance strategies for approaching corners

Performance Results

The implemented solution successfully exceeded the competition's minimum lap time requirements while maintaining robust obstacle avoidance capabilities. The optimization of the racing line through gradient descent, combined with the sophisticated braking system, allowed the vehicle to maintain higher cornering speeds while remaining responsive to dynamic obstacles.
